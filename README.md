# Arena v 2.0
 A new version of Arena

В игре используются следующие механизмы:
1. Наследование
2. Полиморфизм
3. Агрегирование
4. Шаблонные функции
5. Шаблонные классы
6. Абстрактные базовые классы
7. Исключения
8. Указатели на функции
9. Указатели на методы классов
10. Многопоточное программирование, поверхностно 
11. Умные указатели
12. STL
13. Лямбда функции
